# TodayHIT QQ 命令参考（缇安版）

所有查询结果以**合并转发消息卡片**输出，每条包含标题、时间、部门、来源、链接，单次最多 50 条。

| 来源 | 说明 |
|------|------|
| 今日哈工大 | today.hit.edu.cn 公告、快讯 |

| 课程资料 | GitHub HITCS 仓库更新 |
